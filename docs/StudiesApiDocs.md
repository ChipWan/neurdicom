# Study Endpoint

**Base URL:** /api

| Resource  | Description  |
| ------------ | ------------ |
| GET /studies  | Find all patients  |
| GET /studies/:id  | Find patient by id  |
| GET /studies/:id/series  | Find studies for patient  |
| PUT /studies/:id  | Update study  |
| DELETE /studies/:id  | Delete study  |
