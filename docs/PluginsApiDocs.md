# Plugins API

**Base URL:** /api

| Resource  | Description  |
| ------------ | ------------ |
| GET /plugins  | Find all plugins  |
| GET /plugins/:id  | Find plugin by id  |
| POST /plugins | Upload plugin  |
| DELETE /plugins/:id  | Delete plugin  |
