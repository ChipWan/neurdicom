# Patient endpoint

**Base URL:** /api

| Resource  | Description  |
| ------------ | ------------ |
| GET /patients  | Find all patients  |
| GET /patients/:id  | Find patient by id  |
| GET /patients/:id/studies  | Find studies for patient  |
| PUT /patients/:id  | Update patient  |
| DELETE /patients/:id  | Delete patient  |
