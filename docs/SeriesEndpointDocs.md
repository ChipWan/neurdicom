# Series Endpoint

**Base URL:** /api

| Resource  | Description  |
| ------------ | ------------ |
| GET /series  | Find all series  |
| GET /series/:id  | Find series by id  |
| GET /series/:id/instances  | Find series' instances  |
| PUT /series/:id  | Update series |
| DELETE /series/:id  | Delete series |
